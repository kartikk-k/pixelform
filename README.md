# Pixelform

A grid-based SVG painting tool. Click to paint cells, choose a pattern style (rounded, square, blob, leaf, chamfer), and export your creation as a clean SVG.

## Features

- Multiple pattern styles with connected cell rendering
- Symmetry modes (horizontal, vertical, both)
- Pan, zoom, and pinch-to-zoom support
- Undo/redo
- SVG export
- Shareable URLs with encoded state

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start painting.
